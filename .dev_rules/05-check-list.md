## 11. Итоговые чек‑листы

### Чек‑лист перед PR
- [ ] Нет лишних `"use client"`.
- [ ] Все цвета — только через CSS‑переменные.
- [ ] Все новые строки переведены (en/ru).
- [ ] Ключи в списках — не индекс (используйте `crypto.randomUUID()` или `item.id`).
- [ ] Изображения через `next/image`.
- [ ] Server Actions типизированы через Zod.
- [ ] Нет `any` (или есть комментарий).
- [ ] Проверена базовая доступность (aria, фокус).
- [ ] Компонент обёрнут в `Suspense`, если грузит данные.

### Чек‑лист перед продакшеном
- [ ] `error.tsx`, `global-error.tsx`, `not-found.tsx` присутствуют.
- [ ] Настроены `sitemap.ts` и `robots.ts`.
- [ ] Проверены Core Web Vitals (Lighthouse ≥ 90).
- [ ] Настроена аналитика и Sentry.
- [ ] Переменные окружения корректно разделены (публичные/приватные).
- [ ] Все секреты в CI/CD зашифрованы (GitHub Secrets).
- [ ] Написаны тесты на критические пути.
- [ ] Проведён аудит зависимостей (`npm audit`).
